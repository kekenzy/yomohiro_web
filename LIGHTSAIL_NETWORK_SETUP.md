# AWS Lightsail ネットワーク設定ガイド

## 🔒 HTTPS（ポート443）を有効にする手順

HTTPSでアクセスするには、AWS Lightsailのネットワーク設定でポート443を開く必要があります。

### 1. AWS Lightsailコンソールでポート443を開く

1. [AWS Lightsail Console](https://lightsail.aws.amazon.com/)にアクセス
2. インスタンス（`yomohiro-web`など）を選択
3. 「ネットワーキング」タブをクリック
4. 「ファイアウォール」セクションで「ルールを追加」をクリック
5. 以下の設定でルールを追加：
   - **アプリケーション**: カスタム
   - **プロトコル**: TCP
   - **ポート範囲**: 443
   - **ソース**: 任意の場所（0.0.0.0/0）
   - **説明**: HTTPS
6. 「保存」をクリック

### 2. 既存のルールを確認

以下のポートが開いていることを確認：
- **ポート22**: SSH（既に開いているはず）
- **ポート80**: HTTP（既に開いているはず）
- **ポート443**: HTTPS（追加が必要）

### 3. 設定確認

ポート443が開いたら、以下で確認：

```bash
# ローカルからテスト
curl -k -I https://18.183.91.131/

# ブラウザでアクセス
# https://18.183.91.131
```

### 4. 注意事項

- 自己署名証明書のため、ブラウザで警告が表示されます
- 「詳細設定」→「続行」でアクセスできます
- 本番環境では、Let's Encryptの証明書を使用することを推奨します

## 📋 現在の設定状態

- ✅ Nginx: ポート443でリッスン中
- ✅ SSL証明書: 自己署名証明書を生成済み
- ✅ Django設定: HTTPSを有効化済み
- ⚠️ Lightsailネットワーク: ポート443を開く必要があります
